### Autentificacion con JSON Web Tokens
** REQUISITOS **
- Node js y npm

# Instalacion
1. **Clonar el repositorio**
 ```bash
   git clone https://github.com/Obed-Is/autenticacion-con-jwt-nodejs.git
   ```
2. **Ingresar a la carpeta del proyecto**  
   Naveguen a la carpeta llamada 'AP' que se ha creado:  
   ```bash
   cd AP
   ```
3. **Instalar dependencias**  
   En la terminal del proyecto ejecutar
   ```bash
   npm install
   ```
4. **Ejecutar el proyecto**  
  Una ves instalado correctamente prueden ejecutar en la terminal el siguiente comando para comenzar a usarlo
   ```bash
   npm run dev
   ```
